Below is a high-level scaffold for a modern Angular “Configuration Hub” application using the PFM UI/Layout libraries. It gives you: • Top‐level shell (AppModule / AppComponent) • Side nav + header + breadcrumbs + footer via pfm-layout • Angular routing into six feature areas • A shared WizardOverlayComponent for all create/update flows • Stubs for each of the six sections (Tenants, Form Registry, Resource Library, Cohorts, Resource Bundles, Deliver Bundles) • Table + New/Delete/Search boilerplate in each feature component • Example of opening the wizard to Create/Edit You can clone this into an Angular CLI project (`ng new config-hub --routing`) and then install the pfm packages you need (`npm install pfm-ui pfm-layout pfm-ui/components pfm-ui/controls …`).
